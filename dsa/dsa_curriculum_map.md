## 🎓 Pedagogical Design Principles
1. **Analogies first:** Every lesson starts with real-world descriptions of the concepts.
2. **Encapsulation & OOP:** Implementations use object-oriented Python with type hints to teach clean code.
3. **No magic lines:** All steps, loops, and index expansions are clearly commented.
4. **Active Practice:** Each lesson contains exercises and a larger mini-challenge to test student comprehension.
